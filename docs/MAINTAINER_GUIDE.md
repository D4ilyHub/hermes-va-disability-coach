# Maintainer guide

## Release baseline

- Repository version: 1.0.0
- Hermes verified: 0.18.2
- Research cutoff: 2026-07-16
- Registry: 339 sources / 68 domains / 291 official-primary / 55 clinical

## Source correction

Confirm the exact proposition, authority class, effective/revision date, supersession, and affected runtime files. Update both registry copies, `source_metadata.json`, citations, contradiction/freshness logs, and tests. Never replace controlling law with a convenient summary.

## Rating change

Run the user-initiated source staging command, compare official XML and Federal Register rule, determine effective date, update only affected snapshot records, add historical limitation notes, run estimator and regression tests, and record the change. Do not automatically promote staged XML.

## Hermes compatibility

Before release, verify frontmatter, tap layout, allowlisted support directories, explicit file references, security scanning, install, check/update/uninstall behavior, and latest tagged Hermes version. Raise the compatibility range only after testing.

## Privacy review

Search for names, addresses, service numbers, SSNs, phone/email, credentials, tokens, and realistic medical narratives. Examples must remain synthetic and generic.
