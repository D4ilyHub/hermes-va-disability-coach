# Changelog

All notable changes follow Semantic Versioning.

## [1.0.0] - 2026-07-16

### Added

- Public, persona-neutral Hermes skill with 19 routed operating modes.
- Direct, collaborative coaching posture that separates hard safety/integrity boundaries from recommendations and user decisions.
- Concrete recommendation pattern with explicit rationale, tradeoffs, conditional branches, and user agency.
- Context-aware blind-spot screening that uses occupational and service patterns as question generators rather than diagnostic or nexus conclusions.
- Clinician-conversation support for reasonable evaluation and treatment options without prescribing or directing care.
- Rating-aware evidence prioritization without target-rating manipulation, plus a correction protocol that replaces rejected assumptions and re-runs affected reasoning.
- De-identified coaching regression scenarios covering over-rigid refusals, treatment discussions, occupational-pattern misuse, ordinary uncertainty, and user corrections.
- Active-duty, BDD, SHA, Guard/Reserve, IDES/MEB/PEB, care, evidence, C&P, review, and representation references.
- Consent-based sensitive communication and separate urgent/crisis workflows.
- De-identified issue schema and Markdown/YAML/CSV templates.
- Partial, versioned rating-schedule snapshot with provenance and stale-data gate.
- Deterministic combined-rating and bilateral-factor calculator, including § 4.26(d) optimization.
- Date-relative transition planner, condition comparison, freshness check, user-initiated source staging, and bundle validator.
- Curated source registry, research logs, threat model, maintainer documentation, CI, and substantial scenario tests.
- Publication-ready GitHub banner, final repository coordinates, and an explicit independent-project disclaimer.
